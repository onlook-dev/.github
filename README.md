### Open source

* [`onlook-dev/onlook`](https://github.com/onlook-dev/onlook) is the open-source
visual editor that started Onlook.
* [`onlook-dev/desktop`](https://github.com/onlook-dev/desktop) is the open-source
local IDE for visually editing Tailwind codebases.
* [`onlook-dev/chrome-extension`](https://github.com/onlook-dev/chrome-extension) was the open-source
Chrome Extension for visually editing codebases in the browser.
